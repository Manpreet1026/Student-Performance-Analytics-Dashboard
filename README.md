# Student Performance Analytics Dashboard

## Project Overview

This project analyzes student performance data to identify academic trends, performance patterns, and factors that influence student success.

The dataset was cleaned and prepared in Excel, and an interactive Tableau dashboard was developed to visualize key performance metrics and support data-driven decision-making.

---

## Business Objective

The objective of this project is to explore student performance data and uncover insights related to academic achievement, engagement, and overall performance trends.

The project analyzes:

- Student performance trends
- Academic achievement patterns
- Student engagement metrics
- Performance comparisons across groups
- Key factors influencing student success

## Key Insights

- Student engagement showed a positive relationship with academic performance.
- Higher-performing students demonstrated more consistent performance across evaluation metrics.
- Performance trends varied across student groups and categories.
- Interactive visualizations helped identify areas requiring academic support.
- Data-driven insights can support educational planning and student success initiatives.
